# Mamon

## Initalize new codespace:


### Install npm in vue folder:
```cmd
cd client
npm i
```
### Install npm in sanity folder:
```cmd
cd functions
npm i
```

## Steps to run in development:
### Start Sanity:
```cmd
cd functions
npx @sanity/cli start
```
### Start Vue:
```cmd
cd client
npm run dev
```

## Deploy:
Set up Pull Request to ```main```
